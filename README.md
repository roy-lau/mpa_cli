# mpa_cli

### 为什么做？

> 现在前端虽然是spa(单页应用)的天下了。但是,还有一些固执的公司还在用多页。所以mpa-cli横空出世了！

### 做成什么样？

1. 将HTML从开发目录移动到生产目录
2. 如果用的是less/sass/styl等将其编译成css，如果用的是css对其压缩并自动加入浏览器前缀
3. 压缩打包图片和字体图标资源
4. 压缩混淆js,将新的ES6/7语法转化成ES5
5. 使用热更新,文件保存后页面自动更新
6. 做个反向代理,解决跨域问题
